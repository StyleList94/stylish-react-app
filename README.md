# Stylish React App

Quick start your React Application

## Featured

- Vite React Typescript
- SWC
- Vitest
- Testing Library
- ESLint Airbnb rules
- Prettier

## Get Started

### Use template

```bash
npx degit StyleList94/stylish-react-app stylish-app
```

### Install Dependencies

```bash
npm install

# or yarn
yarn install
```

### Run App

```bash
npm run dev

# or yarn
yarn dev
```

### Testing

#### CI/watch

```bash 
npm run test

# or yarn
yarn test
```

#### UI Mode

```bash 
npm run test:ui

# or yarn
yarn test:ui
```

#### Coverage

```bash 
npm run test:coverage

# or yarn
yarn test:coverage
```
